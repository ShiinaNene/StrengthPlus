# StregnthPlus

#### 介绍
A JavaPlugin by Paper-Spigot API for MineCraft 

#### 软件架构
软件架构说明
    undefind

#### 安装教程

1.  一台至少支持java8以上的的计算机
2.  将StrengthPlus 插件安装到Plugin目录里
3.  配置StrengthPlus 文件夹，配置config.yml文件

#### 使用说明

1.  将插件安装至服务器中，确保插件yml文件正常
2.  自行更改yml文件，并调试，确保服务器中公平，正常运行
3.  结束
